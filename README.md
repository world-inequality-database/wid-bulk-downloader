# Bulk-downloader
This repository automates easy access to [World Inequality Database (WID.world)](https://wid.world/) data, processes and cleans the selected series based on the countries and indicators specified by the user. 

The script is located in the World Inequality Database folder, which should be downloaded from the official WID GitHub repository. A detailed description of the folder structure is available in READ.ME files.
What does the script do? It automates access to WID data, processes and cleans the selected series based on the countries and indicators specified by the user. The cleaned datasets are saved in the Cleaned WID Data folder.

Check the READ.ME files before using this tool.
